# OpenTelemetry Utilities for Haskell Exceptions

[![hs-opentelemetry-utils-exceptions](https://img.shields.io/hackage/v/hs-opentelemetry-utils-exceptions?style=flat-square&logo=haskell&label=hs-opentelemetry-utils-exceptions&labelColor=5D4F85)](https://hackage.haskell.org/package/hs-opentelemetry-utils-exceptions)

This package provides utilities for using OpenTelemetry with the `exceptions` package.

See https://github.com/yesodweb/yesod/issues/1533 for discussion on some subtle issues that may arise from using `MonadMask`.
